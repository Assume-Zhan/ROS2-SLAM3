# ROS2 ORB-SLAM3

A repo build a docker image with ros2 humble and orb-slam3

### Build and Run

```bash
git clone https://github.com/Assume-Zhan/ROS2-SLAM3.git ros2_orbslam_docker
```

```bash
cd ros2_orbslam_docker
./run.sh
```
